# CHRONICLE OMEGA V1.0

## MISSION

Build **CHRONICLE**, a modern, production-ready global news platform using **only HTML, CSS, and Vanilla JavaScript**.

Do **NOT** use:

* React
* Next.js
* Vue
* Angular
* Svelte
* Bootstrap
* Tailwind
* jQuery

The website must feel as polished and professional as BBC, CNN, Reuters, or The Verge while remaining a pure HTML/CSS/JavaScript project.

---

# TECH STACK

Frontend:

* HTML5
* CSS3
* Vanilla JavaScript (ES2022)

Backend:

* Supabase

Use Supabase for:

* Database
* Authentication
* Storage
* Realtime updates

---

# DESIGN

Create a premium news experience.

Requirements:

* Fully responsive
* Mobile-first
* Fast loading
* Clean animations
* Glassmorphism where appropriate
* Smooth transitions
* Dark mode
* Light mode
* Professional typography
* Accessible colors
* SEO-friendly HTML
* Lazy loading
* Modern cards
* Beautiful hero section

No placeholder styling.

Everything should look production-ready.

# SECURITY

Use Supabase Authentication.

Only admins may:

Create articles

Edit articles

Delete articles

Manage categories

Manage comments

Manage media

Manage settings

Protect every admin route.

Never expose service-role keys.

Use Row Level Security (RLS).

Validate all inputs.

Escape HTML to prevent XSS.

Use parameterized queries where applicable.

# CODE QUALITY

Use modern ES2022 JavaScript.

Separate concerns.

No duplicated code.

Write reusable functions.

Use async/await.

Use modules.

Comment complex logic only.

Keep the project organized and maintainable.

---

# FINAL GOAL

Build a complete, production-quality news platform that is ready to deploy to Netlify with Supabase as the backend.

The finished project should feel like a commercial news website rather than a beginner project, with excellent performance, clean architecture, polished UI, strong security, and room to scale.
